# Secret Code Translator

A fun Python project that converts normal text into a secret coded message and can also decode it back to the original text.

This project is great for beginners to understand string manipulation, loops, and basic logic building in Python.

---

## Features

* Encode normal message into secret code
* Decode secret code back to original message
* Custom logic-based transformation
* Fast and lightweight
* Beginner-friendly project

---

## Tech Stack

* Python

---

## Project Structure

```
secret-code-translator/
│── main.py
│── README.md
```

---

## How It Works

1. User enters a message
2. Choose:

   * `1` → Encode
   * `0` → Decode
3. Program processes each word and applies secret logic
4. Output is displayed instantly

---

## Usage

Run the program:

```
python main.py
```

Example:

```
Enter your message: hello world  
Press 1 for coding or 0 for decoding: 1  

Output: (secret coded message)
```
